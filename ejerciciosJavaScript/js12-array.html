Con el uso de la función every, que recibe como argumento una función, comprobamos si los elementos de un array satisfacen la condición codificada en la función parámetro. En este caso, comprobamos que los elementos del array anterior sean pares. En este caso son todos pares y se devuelve "true".
Repetimos el procedimiento sobre el array [2,26,38,12,7,8] y pasando la función parámetro como función anónima. En este caso hay un elemento que no satisface la condición. Como not every elemento del array la cumple, every devuelve "false".
