Con el uso de la función some, que recibe como argumento una función, comprobamos si alguno de los elementos de un array satisface la condición codificada en la función parámetro. En este caso, si algún elemento del array [9,37,15,4,4,11] es par. Hay dos elementos que si lo son, por tanto some devuelve "true".
Si eliminamos uno de los elementos [9,37,15,4,11], some sigue devolviendo true. En el momento que no queda ningún elemento par, some devuelve false.
